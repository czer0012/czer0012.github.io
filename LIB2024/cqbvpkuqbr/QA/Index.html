  2. Question: Define version control and give one example.

    Version Control: a system that keeps track of all changes made to a file or project. Useful for referring to previous versions of your code so nothing gets lost or corrupted. Version control also protects against incompatible changes to a file, if edits react in unexpected ways, a file free of those changes still exists. In class we will be using GitHub.

  3. Question: What tag is used to create the title of a web page? Give an example.

    The <title> tag. The text that is nested in this tag will be displayed in the browser tab.

    For this page, the "title" looks like this: <title>Q&A - LIB2024 - Erin Czerneda</title>

  4. Question: What tag is used to create a paragraph in html? Give an example.

    The <p> tag. This will start a new line of text.
